# Freakyville-Datahub
Offentlige data om Freakyville vi benytter os af i vores projekter.

> [!IMPORTANT]
> Alle disse data kan findes af alle og er som udgangspunkt offentlige. Dataene opbevares i forskellige formater så som csv, tsv og JSON.
>
> Hvis du vil bidrage med data eller har behov for det ifm. med en udvidelse du arbejder på til [FreakyvilleX](https://github.com/WildTooth/FreakyvilleX),
> skal du oprette en pull request hvor du forklarer hvad det er og hvad du har af planer for at integrere det med addonnet.
> 
> Dataene kan benyttes af alle og vi modtager gerne bidrag, også hvis det ikke er med henblik på at det skal benyttes i FreakyvilleX.
> Dvs. hvis du laver dit eget projekt og du gerne vil gemme nogle offtenlige data sammen med nogle af dem vi benytter os af, skal du være velkommen til at oprette en pull request hvor du forklare dette.
